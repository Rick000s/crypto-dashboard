# 🚀 Crypto-Dashboard Pro

A modern, high-performance trading terminal for real-time crypto asset management. This dashboard combines deep functionality with a professional adaptive design and multi-theme support.

## 🌟 Key Features

* **Multi-Currency & Multi-Language Support**: Full localization for English (EN), Ukrainian (UK), and Czech (CS). View market prices in USD ($), EUR (€), or CZK (Kč).
* **Three Unique Themes**: 
    * **Default**: A classic dark interface for comfortable daily use.
    * **Midnight**: A deep black "OLED-friendly" theme for maximum focus.
    * **Light**: A high-contrast, professional light-grey design optimized for readability.
* **Live Interactive Charts**: Powered by Chart.js with adjustable time intervals ranging from 15 minutes to 31 days.
* **Virtual Portfolio System**: Buy and sell assets virtually, track your "Available Balance," and monitor your automated transaction history.
* **Favorites List**: Bookmark your top assets for instant tracking and quick access.

## 🛠 Technical Stack

* **HTML5 / Tailwind CSS**: Clean semantic structure with a centralized `style.css` for advanced theme management.
* **Vanilla JavaScript**: Lightweight, framework-free logic for fast execution and low memory footprint.
* **CoinGecko API**: Integration with industry-leading data providers for real-time prices, volumes, and 24h changes.
* **Chart.js**: Dynamic visualization of market trends and historical data.
* **LocalStorage**: Persistent storage for user preferences (language, currency, theme) and portfolio data.

## 📁 Project Structure

* `index.html` — Main application structure and modal windows.
* `style.css` — Centralized theme control using CSS variables and adaptive styling.
* `app.js` — Core logic for API integration, transaction processing, and localization.
