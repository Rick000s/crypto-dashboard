🚀 Crypto-Dashboard Pro (v2.1)
A modern, high-performance web application for real-time cryptocurrency tracking and trading simulation. Powered by the CoinGecko API, featuring multi-currency support and deep personalization.

⭐ Favorites System: Add coins to your personal watchlist for instant access directly on the Home screen.

🖼️ Profile Personalization: Update your username and set a custom avatar using a direct image URL.

📅 Standardized Charts: Chart dates are now strictly formatted in en-US for better readability and a professional look.

🔄 Live Portfolio Updates: Asset cards in the Portfolio view now refresh immediately after every trade (Buy/Sell) without needing a page reload.

🛠 Core Features
Real-Time Trading: Simulate buying and selling assets at live market prices with a virtual balance.

Dynamic Interactive Charts: Powered by Chart.js, featuring intervals from 15 minutes up to 31 days.

Multi-Currency Support: View prices and balances in USD ($), EUR (€), or CZK (Kč).

Global Localization: Fully localized UI in English, Ukrainian, and Czech.

Local Persistence: All user data (balance, portfolio, favorites, and settings) is securely stored in localStorage.

📦 Technical Stack
Frontend: HTML5, Tailwind CSS (for modern, responsive styling).

JavaScript: Vanilla JS (ES6+) for logic and state management.

Data Visualization: Chart.js.

API Integration: CoinGecko (Markets, Search, and Market Chart endpoints).
